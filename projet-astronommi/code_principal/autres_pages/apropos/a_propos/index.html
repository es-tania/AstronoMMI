A propos du site

Ce site est le fruit d’un projet tutoré de première année du DUT Métiers du Multimédia et de l’Internet (MMI). Le projet consiste à concevoir un site web présentant les métiers possibles après avoir fait un DUT MMI, en réinvestissant toutes les compétences acquises au cours de cette première année. Ce projet nous a donc permis de mettre en pratique tout ce que nous savions faire depuis le début de cette première année de DUT.
Pour la réalisation de ce projet, nous avons alors eu besoin de maîtriser le HTML et le CSS pour la mise en forme de la page, les cours de base de données afin de concevoir une base de données efficace et pertinente, mais aussi du PHP pour créer un formulaire de contact fonctionnel. L’utilisation des logiciels pour le graphisme est également nécessaire pour créer tous les visuels de notre site. Le choix du thème de notre site, à savoir la “galaxie MMI” a permis aux graphistes du groupe d’exprimer leur créativité à travers les visuels du site. C’est aussi une manière divertissante de présenter nos fiches métier. Ainsi, vous pouvez retrouver sur notre site des fiches métier complètes tout en effectuant une balade sympathique à travers les différents univers graphiques.
